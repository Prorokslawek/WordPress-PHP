== Description ==
This is a easy, and child-friendly Plugin to help learning the Past simple form.

== Instalation ==
1. In your admin panel, go to plugins ->click the 'Add New' button.
2. Type in english-table in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Install now' button.
